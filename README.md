# 插件：战弟风云 (BattleDick)

一个基于 AstrBot 的斗鸡插件，增长你的dick，然后与群友战斗！

## 命令列表

| 命令 | 权限 | 说明 |
|------|------|------|
| `/growth` | 群聊 | 每日成长，随机增加 0.1~5.0 cm 长度 |
| `/pvp <赌注>` | 群聊 | 发起一场赌注为指定长度的决斗（赌注必须为正数） |
| `/斗鸡` | 群聊 | 响应当前群内正在进行的挑战 |
